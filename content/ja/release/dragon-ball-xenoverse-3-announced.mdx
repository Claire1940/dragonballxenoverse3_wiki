<Section icon={Swords} title="進化した戦闘システム：ソウルアシストとソウルスイッチ">

**『ドラゴンボール ゼノバース３』**の戦闘は、シリーズの特徴であるハイスピードな3Dアクションを継承しつつ、2つの主要な新システムによって戦術的な深みが加わっています。これらのメカニクスは、カスタムアバターを主役にしつつも、シリーズのレジェンドキャラクターたちとの繋がりをより強く感じられるように設計されています。

<Steps>
  <Step num="1" title="アバターカスタマイズ">
    サイヤ人や地球人など、様々な種族からヒーローを作成します。選んだ種族によって、基本コンボや変身形態が大きく変化します。
  </Step>
  <Step num="2" title="探索と移動">
    西の都の空を飛び回り、新しいブースト機能を駆使して、広大なバトルフィールドをシームレスに駆け巡ります。
  </Step>
  <Step num="3" title="ソウルアシストの発動">
    **ソウルアシスト**を発動すると、仲間の戦士を呼び出して強力な連携攻撃を繰り出せます。師匠となるNPCとの絆を深めることで、より強力な恩恵が得られます。
  </Step>
  <Step num="4" title="ソウルスイッチのマスター">
    **ソウルスイッチ**システムでは、伝説の戦士の魂の力を一時的に宿すことができます。これにより、その戦士特有の技や大幅なステータスアップが可能になります。
  </Step>
</Steps>

ソウルスイッチは特に注目すべき新要素で、戦闘中にプレイスタイルを切り替えることができます。異なる「ソウル」を装備することで、様々なタイプの敵に適応できるようになります。

| ソウルスイッチの例 | 戦闘でのメリット |
| :--- | :--- |
| **ベジータのソウル** | 攻撃的な気功波と高いプレッシャーに特化 |
| **ピッコロのソウル** | 防御性能の向上とリーチの長い攻撃 |
| **未来トランクスのソウル** | 素早い剣撃と機動力に特化 |
| **クリリンのソウル** | 戦略的なユーティリティとサポート系の気功攻撃 |
| **天津飯のソウル** | ハイリスク・ハイリターンの精密な一撃 |

<Callout type="tip" title="戦略的なスイッチ">
ソウルスイッチは単なるパワーアップではありません。技セットそのものが変化します。多数の敵に囲まれた時は**ピッコロのソウル**で範囲攻撃を行い、強力なボスを仕留める時は**ベジータ**に切り替えるといった戦略が可能です。
</Callout>

</Section>

<Section icon={Sparkles} title="参戦確定キャラクターリスト">

| :--- | :--- | :--- |

| **ブルマ** | **NPC** | 西の都でのストーリーガイド |
| **ガンマ1号** | **プレイアブル/仲間** | グレートサイヤ隊のリーダー格 |
| **ブレット** | **新オリジナル** | 西の都の住人 / ストーリーキャラ |
| **リリカ** | **新オリジナル** | 西の都の住人 / ストーリーキャラ |
| **ロム** | **新オリジナル** | 西の都の住人 / ストーリーキャラ |
| **タップ** | **新オリジナル** | 西の都の住人 / ストーリーキャラ |

<Callout type="warning" title="ロースターの噂について">
非公式サイトで見られる「全キャラ流出」などの情報には注意してください。現時点で確定しているのは、上記のキャラクターとソウルスイッチに関連する戦士（クリリン、ピッコロ、ベジータ、未来トランクス、天津飯）のみです。
</Callout>

**ガンマ1号**の参戦は、映画『ドラゴンボール超 スーパーヒーロー』のエピソードがグレートサイヤ隊の活動において重要な役割を果たすことを示唆しています。また、ブレットやリリカといった「初登場のオリジナルキャラクター」たちは、大きな戦いが始まる前の、ドラゴンボールの世界におけるより身近で市民的な視点を描くことを予感させます。
