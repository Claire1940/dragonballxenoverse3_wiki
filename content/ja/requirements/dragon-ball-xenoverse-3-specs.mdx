<Section icon={Swords} title="ドラゴンボール ゼノバース３ 目標PCシステム要件">

最新のトレーラーで見られるようなビジュアルの忠実度を実現するには、『ゼノバース２』から大幅なハードウェアのアップグレードが必要です。**ドラゴンボール ゼノバース３のスペック**は、最新のPCアーキテクチャを活用するように設計されており、特にGPUメモリとSSDのスループットに重点を置いています。AGE 1000の西の都をシームレスなオープンワールドとして探索するため、従来のHDDでは飛行中や高速ブースト中に大きなスタッタリング（カクつき）が発生する可能性が高いです。

以下のデータは、2027年のローンチに向けた公式プラットフォームとの互換性およびエンジン要件に基づく目標技術仕様です。

| コンポーネント | 最低要件 | 推奨仕様 |
| :--- | : :--- | :--- |
| **OS** | Windows 10 (64-bit) | Windows 11 (64-bit) |
| **プロセッサ** | Intel Core i5-10600 / AMD Ryzen 5 3600 | Intel Core i7-12700 / AMD Ryzen 7 5800X |
| **メモリ** | 16 GB RAM | 32 GB RAM |
| **グラフィック** | NVIDIA GTX 1660 Ti / RX 5600 XT | RTX 3070 / RX 6800 XT |
| **DirectX** | Version 12 | Version 12 (Ultimate) |
| **ストレージ** | 100 GB の空き容量 (SSD必須) | 100 GB NVMe M.2 SSD |

**ソウルスイッチ**および**ソウルアシスト**機能の導入により、ゲームは複数のキャラクターアセットを同時にメモリに保持する必要があります。これが、メモリの最低ラインが16GBである主な理由です。旧来の8GBシステムを使用しているプレイヤーは、アセットのポップインや戦闘の遅延に悩まされることになるでしょう。

<Callout type="warning" title="ハードウェアに関する警告">
『ドラゴンボール ゼノバース３』はレガシーハードウェアをサポートしません。ローンチ時のシェーダーコンパイルの問題を避けるため、お使いのGPUが最低でもDirectX 12 Feature Level 12_1をサポートしていることを確認してください。
</Callout>
