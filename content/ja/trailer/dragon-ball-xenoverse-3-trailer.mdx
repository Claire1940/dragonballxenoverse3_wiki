<Section icon={Map} title="AGE 1000と新生「西の都」の探索">

**ドラゴンボール ゼノバース３**の物語の核心は、全く新しい時代である**AGE 1000**へとプレイヤーを誘います。この時代へのジャンプにより、テクノロジーと武術が共に進化したドラゴンボールの世界を新鮮な視点で捉えることができます。トレーラーでは、西の都が単なる静的なメニューや小さな拠点ではなく、探索とストーリー進行の主要な舞台となる、高度に発展した活気ある大都市として描かれています。

この時代でプレイヤーは、鳥山明氏独特の芸術的タッチで命を吹き込まれた世界に足を踏み入れます。街には西の都大学やグレートサイヤ隊の本部があり、プレイヤーが作成した主人公は物語の早い段階でこの部隊に加わることになります。この変化は、『ゼノバース2』のコントン都と比較して、より「生きている世界」へのアプローチを示唆しています。

<Section icon={Swords} title="戦闘メカニズム：ソウルアシストとソウルスイッチ">

**ドラゴンボール ゼノバース３**の戦闘は、シリーズ特有のハイスピードなアクションを維持しつつ、プレイヤーと伝説の英雄たちとの絆を強調する戦術的な要素を導入しています。トレーラーでは、2つの革新的なシステム「**ソウルアシスト**」と「**ソウルスイッチ**」が紹介されました。これらのメカニズムは、「カスタムヒーロー」が作品の広大なパワーバランスにより深く組み込まれていると感じられるように設計されています。

**ソウルアシスト**は、プレイヤーが強力な仲間を呼び出し、連携攻撃を行うことを可能にします。これは単なるアシスト技ではなく、困難な戦況を一変させることができるシネマティックなチームアップ攻撃です。一方、**ソウルスイッチ**は変身メカニズムです。プレイヤーのアバターが一時的にクラシックキャラクターの「ソウルパワー」を宿し、一定時間、固有の技セットや強化されたステータスを使用できるようになります。

| メカニズム | 機能 | 戦闘への影響 |
| :--- | :--- | :--- |
| **ソウルアシスト** | 仲間との連携攻撃 | 高い瞬間ダメージとコンボの延長 |
| **ソウルスイッチ** | 一時的なソウルの憑依 | 固有スキル（例：ファイナルフラッシュ）へのアクセス |
| **気功攻撃** | エネルギーベースの攻撃 | 遠距離からのプレッシャーと象徴的な必殺技 |
| **ハイスピードアクション** | 移動とブースト | 回避と戦闘の勢いの維持 |

トレーラーでは、ソウルスイッチシステムを通じて利用可能なソウルを持つ数名のキャラクターが具体的に挙げられています。これにより、プレイヤーは自分のカスタムキャラクターの基本ステータスや種族をどの英雄のソウルが補完するかを決定するという、「ビルド構築」の奥深さが加わります。

<Steps>
  <Step num="1" title="ヒーローを作成する">
    種族（サイヤ人、地球人など）を選び、外見をカスタマイズして戦闘の基礎を固めます。
  </Step>
  <Step num="2" title="ハイスピードな戦闘を展開する">
    気功攻撃や近接攻撃を駆使して相手を弱らせます。
  </Step>
  <Step num="3" title="ソウルアシストを発動する">
    ドラゴンボールの仲間を呼び出し、強力な合体エネルギー弾や物理攻撃を叩き込みます。
  </Step>
  <Step num="4" title="ソウルスイッチを起動する">
    ベテラン戦士の魂を宿し、彼らの象徴的な究極技で決着をつけます。
  </Step>
</Steps>

<Callout type="warning" title="戦略的なタイミング">
ソウルスイッチは一時的な状態であるようです。この変身のタイミングをマスターすることが、ハイレベルなプレイやボス戦において非常に重要になります。
</Callout>

</Section>

**ドラゴンボール ゼノバース３ トレーラー**により、登場キャラクターはプレイヤー作成の主人公、シリーズの主要キャラクター、そして全く新しいオリジナルキャラクターの混合になることが確認されました。**ブレット、リリカ、ROM、タップ**の導入は、ゼノバースの物語における大きな拡張を意味します。これらのキャラクターはAGE 1000の設定に合わせて特別にデザインされており、グレートサイヤ隊の中でプレイヤーの仲間やライバルとして活動します。

オリジナルキャラクターが新しいプロットを牽引する一方で、ブルマやガンマ1号といったアイコンの存在が、このゲームがドラゴンボールの正当な続編であることを保証しています。さらに、ソウルスイッチ・メカニズムにより、たとえAGE 1000のストーリーの主役でなくても、幅広いクラシックヒーローたちがゲームプレイの一部として登場します。

| キャラクター | タイプ | 役割 / 勢力 |

| **ガンマ1号** | 味方 | グレートサイヤ隊の連絡員 |
| **ブレット** | オリジナル | AGE 1000の新キャスト |
| **リリカ** | オリジナル | AGE 1000の新キャスト |
| **ベジータ** | ソウルスイッチ | 戦闘変身のソース |

| **ピッコロ** | ソウルスイッチ | 戦闘変身のソース |

ラインナップの多様性により、新しいRPGの方向性を好むファンも、クラシックな対戦格闘要素を愛するファンも、どちらも楽しめる内容となっています。[Steamページ](https://store.steampowered.com/app/1857810/DRAGON_BALL_XENOVERSE_3/)に記載されている「広範なキャラクターカスタマイズ」は、自分だけのヒーローが物語の主役であることを改めて強調しています。
