<Callout type="info" title="見た目とステータスの関係">

自分の好みのプレイスタイルに合った種族を決めるために、以下の詳細な比較表を参考にしてください。

| 種族 | 主な強み | おすすめビルド |
| :--- | :--- | :--- |
| サイヤ人 | 打撃・気弾ともに高い火力 | 打撃/気弾ハイブリッド |
| 地球人 | バランスの良いステータス、技力自動回復 | 基本攻撃＆打撃特化 |
| ナメック星人 | 高いHP、体力自動回復 | 防御型タンク |
| 魔人 | 高い防御力、スタミナ回復速度 | スタミナ重視の防御型 |
| フリーザ一族 | 高い移動速度、気弾ダメージ | 気弾特化スピードスター |

種族を選んだ後は、属性ポイントを正しく配分することが不可欠です。すべてのステータスに平均的にポイントを振るよりも、特定のビルドに特化させる方がはるかに高い効果を得られます。

| ビルドタイプ | 重要ステータス | プレイスタイルの特徴 |
| :--- | :--- | :--- |
| 気弾特化型 | 技力、気弾系必殺技、スタミナ | 遠距離からの高火力攻撃 |
| 打撃特化型 | 最大HP、打撃系必殺技、基本攻撃 | 近接格闘コンボ重視 |
| バランス型ハイブリッド | 技力、スタミナ、基本攻撃 | あらゆる状況に対応可能な汎用性 |

<Callout type="tip" title="ビルドの最適化">
地球人は技力が自動回復するため、チャージのために足を止めることなく頻繁に必殺技を使用できます。扱いやすく安定したキャラクターを求めるプレイヤーに最適です。
</Callout>

<Section icon={Target} title="カスタムキャラクターの育成手順ガイド">

本作の育成要素はアカデミーシステムを中心に構成されています。学校に入学することで、修行パートナーを解放したり、特別な課題に挑戦したり、伝説の戦士たちから指導を受けてキャラクターの潜在能力を試すことができます。

<Steps>
  <Step num="1" title="キャラクター作成">
    種族、性別、体格を選択して、ベースとなるステータスを決定します。
  </Step>
  <Step num="2" title="アカデミーへの入学">
    導入部分のストーリーミッションをクリアして学校に入学し、リラと出会います。
  </Step>
  <Step num="3" title="師匠の選択">
    ハブエリアにいる伝説の戦士たちと交流し、彼らの得意技（必殺技）を習得します。
  </Step>
  <Step num="4" title="覚醒技の解放">
    種族専用の試練をクリアして、キャラクターの主要な変身形態（覚醒技）を解放します。
  </Step>
</Steps>

これらの手順を進める中で、特定のレベルに達すると、より高難度のコンテンツや強力な能力が解放されます。

| レベル帯 | 解放される要素 | おすすめのアクティビティ |
| :--- | :--- | :--- |
| レベル 1-20 | 基本的な師匠、アカデミークエスト | ストーリーモード、パトロールミッション |
| レベル 21-50 | 上級スキル、最初の変身形態 | 師匠の挑戦状、エキスパートミッション |
| Level 51-80 | 覚醒技、ステータス上限解放 | 高難度パラレルクエスト、レイドボス戦 |

<Callout type="warning" title="スタミナ管理">
スタミナのステータスを軽視してはいけません。スタミナ管理を怠ってガードブレイク（スタミナ切れ）を起こすと、相手の究極技に対して完全に無防備になってしまいます。
</Callout>

</Section>

<Section icon={Swords} title="高度なCaC変身形態（覚醒技）の解放">

変身は戦闘における象徴的なシステムです。覚醒技を発動するとキャラクターの姿が変わり、基本攻撃モーションが変化するほか、戦況を一変させるほどのステータス上昇効果を得られます。

| 変身形態 | ステータス上昇効果 | デメリット・弱点 |
| :--- | :--- | :--- |
| 超サイヤ人 | 打撃および気弾ダメージの上昇 | 技力の回復速度が低下 |
| ビースト | 全体的なダメージが大幅に上昇 | 防御力が低下 |
| 潜在能力解放 | バランスの良いステータス上昇、素早い技力チャージ | 発動に多くの技力が必要 |
| 巨大化 | 高いダメージ、無敵フレームの発生 | スタミナが急速に減少 |

どの変身を選択するかは、ビルドによって大きく異なります。例えば、気弾特化型のビルドであれば、技力回復のペナルティが少なく、エネルギー攻撃を強化できる形態が適しています。

<Callout type="success" title="変身のシナジー">
「潜在能力解放」と「地球人」を組み合わせることで、素早い技力生成と安定したダメージにより、敵に絶え間なくプレッシャーを与え続ける高効率なビルドが完成します。
</Callout>

</Section>
