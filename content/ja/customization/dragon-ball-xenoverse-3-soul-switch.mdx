<Section icon={Swords} title="ソウルスイッチ vs ソウルアシスト：メカニズム比較">

ソウルスイッチとソウルアシストの違いを理解することは、AGE 1000でパフォーマンスを最適化しようとするプレイヤーにとって不可欠です。どちらのシステムも伝説の味方が関与しますが、戦術的な目的は全く異なります。ソウルアシストは瞬間的なダメージとコンボの延長のために設計されており、味方が飛び込んで一度の強力な一撃を与えます。対照的に、ソウルスイッチは、一定期間ヒーローの基本的な戦闘行動を変化させる変身状態です。

| 特徴 | ソウルアシスト | ソウルスイッチ |
| :--- | :--- | :--- |
| **メカニズムの目的** | タッグチームのコンボ延長 | 一時的なキャラクター変身 |
| **持続時間** | 瞬間的 / 単発攻撃 | 持続的（メーター制） |
| **ステータスへの影響** | 主人公のステータス変化なし | ソウルに基づいた大幅なバフ |
| **スキルへのアクセス** | 味方が自身のスキルを使用 | 主人公がソウル固有のスキルを獲得 |
| **最適な使用場面** | 敵のガード崩しやコンボのフィニッシュ | 属性やスタイルの不利の克服 |

ソウルスイッチシステムは、戦術的な選択肢を実質的に倍増させます。例えば、一撃の重いサイヤ人ビルドでプレイしている際に、非常に機動力の高いボスに遭遇した場合、未来トランクスのような素早いキャラクターのソウルに切り替えることで、ペースを維持するために必要な敏捷性を得ることができます。この柔軟性は、バンダイナムコが2027年のリリースに向けて実装した、新しいRPG主導の方向性の礎石です。

<Callout type="info" title="戦術的な洞察">
ソウルスイッチは単なるパワーアップではありません。ユーティリティも重要です。一部のソウルは、ベースキャラクターが欠いている防御オーラや回復特性を提供するため、西の都郊外での長期的な生存ミッションには不可欠です。
</Callout>

</Section>

<Section icon={Target} title="ソウルスイッチの流れをマスターする方法">

ドラゴンボール ゼノバース３のソウルスイッチをマスターするには、メーターが満タンのときにボタンを押すだけでは不十分です。戦場を真に支配するには、スイッチを既存のコンボに組み込む必要があります。移行自体にアクティブな当たり判定があるため、タイミングの良いソウルスイッチは、迫りくる攻撃をパリィしたり、途切れてしまうようなジャグリングを延長したりすることができます。

<Steps>
  <Step num="1" title="ソウルメーターを溜める">
    近接戦闘を行い、気功攻撃をヒットさせてゲージを溜めます。バニシングステップの成功やパーフェクトブロックは、このプロセスを大幅に加速させます。
  </Step>
  <Step num="2" title="エントリーポイントを選択する">
    ガードブレイクや重いノックバックを待ちます。ノックバック中にソウルスイッチを発動すると、「ソウル」が出現し、即座に追撃を続けることができます。
  </Step>
  <Step num="3" title="消費速度を管理する">
    発動中、メーターは減少していきます。シグネチャーソウルスキルを使用すると、メーターの消費が早まります。標準攻撃と、1つか2つの強力なソウル能力のバランスをとってください。
  </Step>
  <Step num="4" title="ソウルフィニッシュを発動する">
    メーターが空になる前に、ソウルフィニッシュをトリガーします。これは残りのメーターを消費するシネマティックな究極攻撃で、大ダメージを与え、戦闘をニュートラル状態にリセットします。
  </Step>
</Steps>

</Section>

<Section icon={Shield} title="種族のシナジーとビルドの最適化">

ドラゴンボール ゼノバース３における種族の選択は、ソウルスイッチメカニズムのパフォーマンスに大きな影響を与えます。各種族には固有のステータス傾向があるため、特定のソウルの組み合わせは自然と他のものよりも「メタ」に感じられるでしょう。例えば、バランスの取れたステータスと気の自動回復で知られる地球人は、他の種族と比較してソウルスイッチ状態をより長く維持できます。

| 種族 | 固有の利点 | 推奨されるソウルスイッチのシナジー |
| :--- | :--- | :--- |
| **サイヤ人** | 攻撃力 / ゼンカイボーナス | 最大ダメージのための**ベジータソウル** |
| **地球人** | 気の回復 | 頻繁なアビリティ使用のための**天津飯/クリリンソウル** |
| **ナメック星人** | 体力 / アイテム効果 | 究極の生存能力のための**ピッコロソウル** |
| **フリーザ一族** | 移動速度 | 圧倒的な機動力のための**未来トランクスソウル** |
| **魔人** | 防御 / 回復 | 相手より長く生き残るための**サポートソウル** |

AGE 1000でビルドを構築する際は、「ソウルスロット」をロードアウトの中心として考えてください。もし「ガラスの大砲」のようなサイヤ人ビルドを作るなら、低い体力を補うために防御的なソウルを選ぶか、あるいは高火力のソウルで攻撃をさらに強化して数秒で決着をつけるかを選択できます。Steamおよびコンソール版で確認されている広範なキャラクターカスタマイズオプションにより、全く同じようにプレイする主人公は二人といません。

<Callout type="warning" title="重要な制限">
戦闘中にソウルを切り替えることはできません。装備するソウルは、ミッションに出発する前、またはPvPアリーナに入る前に、グレートサイヤ隊のロードアウト端末で選択する必要があります。
</Callout>

</Section>
