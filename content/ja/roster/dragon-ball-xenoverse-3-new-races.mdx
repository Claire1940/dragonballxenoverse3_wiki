<Section icon={Shield} title="ソウルスイッチとソウルアシストのメカニズム">

ドラゴンボール ゼノバース３の戦闘システムには、**ソウルアシスト**と**ソウルスイッチ**という２つの革命的なメカニズムが導入されています。これらのシステムにより、プレイヤーのカスタム主人公と、ドラゴンボールの世界の伝説的なヒーローたちとの架け橋が築かれます。

### ソウルスイッチの例
ソウルスイッチを使用すると、伝説的なキャラクターのソウルの力を一時的に借りることができ、一定時間、彼らの象徴的な技と大幅なステータス上昇を得ることができます。

| キャラクターソウル | 主な恩恵 | 必殺技 |
| :--- | :--- | :--- |
| **ベジータ** | 攻撃力の向上 | ファイナルフラッシュ |
| **未来トランクス** | 剣ベースのコンボ | バーニングアタック |
| **ピッコロ** | 戦術的防御 | 魔貫光殺砲 |
| **クリリン** | サポート＆ユーティリティ | 気円斬 |
| **天津飯** | 精密な打撃 | 新気功砲 |

<Callout type="success" title="戦闘戦略">
ボス戦や緊迫したPvPの場面でソウルスイッチを活用しましょう。スイッチのタイミングを合わせることで、無敵フレームを利用したり、壊滅的なアルティメット攻撃を繰り出したりして、戦況を一変させることができます。
</Callout>

### ソウルアシストシステム
一時的なスイッチとは異なり、**ソウルアシスト**は永続的な味方の存在です。ソウルアシストを装備することで、味方が一瞬現れて敵を一緒に攻撃する連携攻撃を発動できます。これは、チームワークと集団の強さをテーマにした「グレートサイヤ隊」のコンセプトを強化するものです。

| システム構成要素 | 機能 | 使用方法 |
| :--- | :--- | :--- |
| **ソウルアシスト** | 味方による追撃 | 自動またはトリガー |
| **ソウルスイッチ** | 一時的な変身 | 手動発動 |
| **気弾攻撃** | 主要なエネルギー攻撃 | リソース依存 |
| **ハイスピードアクション** | 回避行動 | スタミナ依存 |

</Section>
