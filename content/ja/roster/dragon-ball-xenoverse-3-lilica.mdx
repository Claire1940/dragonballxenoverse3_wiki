<Section icon={Swords} title="戦闘の進化：ソウルスイッチとソウルアシスト">

**ドラゴンボール ゼノバース３**は、前作までのアクション満載のコアゲームプレイを継承しつつ、2つの革新的なシステム**「ソウルスイッチ」**と**「ソウルアシスト」**を導入しています。これらのメカニクスは、AGE 1000のストーリーで待ち受ける強敵との戦いを制するために不可欠です。

**ソウルスイッチ**は、プレイヤーが一時的に伝説のドラゴンボールキャラクターのソウルパワーを宿すことができるシステムです。これは単なるステータス上昇にとどまらず、一定時間、使用可能なアビリティや戦闘スタイルそのものを変化させます。例えば、ベジータのソウルに切り替えることで「ファイナルフラッシュ」が使用可能になり、ピッコロのソウルに切り替えれば防御性能が大幅に強化されます。

| システム | 説明 | プレイヤーへの恩恵 |
| :--- | :--- | :--- |
| **ソウルスイッチ** | ベテランヒーローのソウルパワーを宿す | 一時的な新しいアビリティとステータス補正 |
| **ソウルアシスト** | ドラゴンボールの仲間を呼び出して共同攻撃を行う | 絶大な瞬間ダメージとコンボの延長 |
| **気弾攻撃** | エネルギー主体の攻撃アクション | 遠距離からのプレッシャーとシネマティックなトドメ技 |
| **高速アクション** | 空中および地上での素早い移動 | 回避行動とアグレッシブな位置取り |

バンダイナムコからの公式情報により、ソウルスイッチシステムを通じて利用できるキャラクターソウルのリストがすでに一部公開されています。これらのキャラクターは、カスタム主人公や**リリカ**たち新キャストのメンター、あるいは「精神的ガイド」としての役割を果たします。

| ソウルキャラクター | 戦闘での役割 | 判明しているアビリティ例 |
| :--- | :--- | :--- |
| **ベジータ** | 圧倒的な攻撃役 | 高ダメージの気弾攻撃 |
| **ピッコロ** | 戦術的なサポート | 防御バリア / カウンター攻撃 |
| **未来トランクス** | スピード / 剣技 | 高速連撃と優れた機動力 |
| **クリリン** | ユーティリティ / サポート | 目くらまし技と素早い気弾 |
| **天津飯** | 精密射撃 / バースト | 集中エネルギー攻撃 |

<Callout type="tip" title="戦略ノート">
ソウルスイッチシステムは、敵がブレイク状態（気絶状態）になったタイミングに合わせるのが最も効果的です。スタン中にベジータなどの高火力ソウルに切り替えることで、ボス戦の戦況を一気にひっくり返すことができます。
</Callout>

</Section>

<Section icon={Target} title="キャラクターロスターと勢力の役割">

**ドラゴンボール ゼノバース３**のロスターは、おなじみの人気キャラクターと、新しい「次世代」キャラクターが融合したものになっています。物語は、プレイヤーのカスタムヒーローがこれら2つのグループとどのように関わっていくかに焦点を当てています。**リリカ**、ブレット、ロム、タップは未来を象徴し、ブルマやガンマ1号といったキャラクターは、これまでのドラゴンボールの世界観との重要な繋がりを提供します。

| キャラクター名 | キャラクタータイプ | 主な所属勢力 | 役割 |
| :--- | :--- | :--- | :--- |
| **カスタムヒーロー** | プレイヤー作成 | グレートサイヤ隊 | 主人公 |
| **リリカ** | オリジナル新規 | AGE 1000の住民 | ストーリーサポート / 仲間 |
| **ブレット** | オリジナル新規 | AGE 1000の住民 | ストーリーサポート / 仲間 |
| **ブルマ** | おなじみのレジェンド | 西の都 / カプセルコーポレーション | ストーリーの導き手 |
| **ガンマ1号** | 再登場のヒーロー | グレートサイヤ隊 | 部隊の指揮官 |
| **ロム** | オリジナル新規 | 不明 | ストーリーキャスト |
| **タップ** | オリジナル新規 | 不明 | ストーリーキャスト |

「グレートサイヤ隊」は、プレイヤーが所属する主要な勢力となります。ガンマ1号の指導のもと、プレイヤーは西の都を守り、AGE 1000の秘密を解き明かすミッションに挑みます。プレイヤーはこれらのミッションを通じて**リリカ**と多くの時間を過ごし、自身の成長とともに彼女のキャラクターアークを深めていくことになります。

<Callout type="warning" title="ロスターの確認">
多くのキャラクターが「ソウルスイッチ」の選択肢として戦闘に登場しますが、従来の形式で全員が完全にプレイ可能であるとは限りません。誰が「カスタマイズ可能なヒーロー」で、誰が「ソウルサポート」キャラクターなのか、常に公式のロスター情報アップデートを確認してください。
</Callout>
